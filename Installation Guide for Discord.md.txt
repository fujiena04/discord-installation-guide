Installation Guide for Discord on Windows

Introduction

Discord is a popular app for creating servers, messaging, and voice chat. This guide will show you how to install Discord on a Windows 10 computer.

Installation types

This article will talk about the installation on Windows, not Mac OS.

System requirements



* Windows 10 or later
* Internet connection
* 100 MB free disk space

Step I: Download the Discord Installer



1. Go to [discord.com](http://discord.com)
2. Click Download or Go to [https://discord.com/download](https://discord.com/download)

Verify installation

Open Discord from the Start menu to confirm it launches correctly.

Upgrade options

Discord automatically updates on its own. No manual action is required.

Downgrade options

Downgrading is not recommended due to security risks.

Uninstallation options



* Go to Control Panel > Programs > Uninstall a program.
